#  Reto-5 Sistema de votación
Implementar un sistema de votación usando `react` y `context`.

## Ejecución

```bash
git clone git@github.com:jocdiazm/makeit-retos.git
cd makeit-retos/Reto-5-Context
npm install
npm start
```
